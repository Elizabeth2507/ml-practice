# ML Interview Practice

This repository contains my hands-on practice for machine learning interview preparation.

## Topics
- Simple Linear Regression
- Multiple Linear Regression
- Logistic Regression
- Decision Trees
- Random Forest
- KNN
- SVM
- Naive Bayes
- Model Evaluation
- Cross Validation
- Regularization


## Create and activate a virtual environment
```
cd ml-practice

python -m venv .venv

source .venv/bin/activate

pip install jupyter numpy pandas matplotlib scikit-learn seaborn

pip freeze > requirements.txt

jupyter notebook
```

## Notebook template
- Problem definition

- Import libraries

- Load or create dataset

- Explore dataset

- Preprocess data

- Split train/test

- Train model

- Predict

- Evaluate

- Plot results
